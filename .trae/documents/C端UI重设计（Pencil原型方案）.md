# 总体风格
- 品牌：李先生的小店（中文界面）
- 设计语言：多巴胺清新风 + 轻拟物卡片，圆角大、阴影浅，不添加等级/勋章/连签等元素
- 网格与尺寸：移动端 390×844 画布（iPhone 14 参考），8px 基准间距，圆角 16px/24px
- 色板：
  - 主色：#ff6b6b（西瓜红）
  - 辅色：#ffa940（琥珀黄）、#409eff（清蓝）
  - 背景：#f5f6fa（浅灰）
  - 文本：#333 主文本；#666 次文本；#999 辅助

# 共用组件（Pencil元件库）
- 顶部胶囊搜索框（含搜索图标）
- 营销大卡（渐变背景、商品图占位、右下角圆形“+”按钮）
- 商品卡（图+名+价+加购按钮）
- 分类方块（双排横向滚动，色块+3D图标位+中文名）
- 状态标签（待核销/已完成/已取消）
- 底部 TabBar（首页/购物车/订单/我的）

# 页面1：首页（Home）
## 顶部
- 标题：李先生的小店
- 搜索：胶囊输入框（占位文案：搜索美味零食…）

## 分类横向滚动（双排）
- 区域高度：两行网格（2×N），每格 120×86，间距 12
- 每格：
  - 背景色块（从色板中分配）
  - 正中 3D风格图标位（占位图片或Emoji），下方中文分类名两行省略
- 末尾：药丸按钮“查看全部”，点击至完整分类树页

## Dopamine Deals（营销卡）
- 轮播两张：左上角角标“店长推荐/热卖”，右下角圆形“+”按钮
- 卡片内居中商品图占位

## 商品列表
- 单列卡片：图 100×100（圆角12）+ 名称 + 次文案“销量 99+” + 价格（¥ 数字加粗）
- 右侧悬浮圆形加购按钮（+）

# 页面2：购物车（Cart）
- 列表项：缩略图 + 名称 + 价格（高亮）
- 数量条：圆形减（-）/加（+）按钮 + 数字
- 底部结算胶囊条：
  - 左：已选件数与合计金额
  - 右：主按钮“去结算”
- 空状态：插画+文案“购物车空空如也”

# 页面3：订单（Order）
- 列表卡片：
  - 左侧竖色条：
    - 待核销=琥珀黄 #ffa940
    - 已完成=绿色 #67c23a
    - 已取消=灰 #c0c4cc
  - 头部：订单号 + 状态标签
  - 内容：商品缩略图集合（+n）、下单时间、副标题（共X件）
- 待核销（状态0）：卡片中段展示“核销码”大号数字（等宽字体），下方“刷新状态”次按钮
- 已完成（状态1）：右侧“去评价”按钮
- 空状态：插画+文案“暂无订单”

# 页面4：我的（Profile）
- 顶部：头像+问候“你好，昵称！”（右下编辑图标）
- 菜单栅格（纵向卡）：
  - 我的订单 → /order
  - 购物车 → /cart
  - 我的评价 → /my-comments
- 编辑资料弹窗（原逻辑保留）：昵称、头像选择（网格六个备选）

# 交互与动效（原生感，但不过度）
- 按钮/卡片点击缩放 0.98，阴影微变（Pencil中用状态页表达）
- 骨架屏：首页商品与订单列表加载态用灰块占位
- 不加入等级/勋章/连签/震动等元素

# Pencil交付结构
- Pages：Home、Cart、Order、Profile、CategoryAll（查看全部分类）
- Components：SearchBar、CategoryTile、ProductCard、PromoCard、OrderItem、TabBar、DialogProfile
- Assets：图标占位（PNG/SVG）、色板样式模板；文本全部中文
- 链接：
  - Home 分类→筛选后的 Home（可用多个状态页）
  - 查看全部→CategoryAll
  - TabBar四个入口互跳
  - 订单卡“去评价”→现有评价入口占位

# 不变更项
- 后端接口与字段不变
- 现有路由结构与逻辑不变（原型仅演示视觉与交互）
