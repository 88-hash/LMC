# 乐逸零食销售系统部署说明文档

## 1. 项目结构更新

为了实现前后端完全分离以及客户端与管理端的独立部署，项目目录结构已进行了重构：

*   **`backend/`**: 后端 Java Spring Boot 项目代码。
*   **`client/`**: 独立的客户端前端项目 (运行在端口 `5173`)。
*   **`admin/`**: 独立的管理端前端项目 (运行在端口 `5174`)。
*   `frontend_backup/`: 原前端项目的完整备份。

## 2. 如何启动系统

### 方法一：桌面快捷方式（推荐）
我们在您的桌面上创建了一个名为 **`LeYi_Start`** 的快捷方式。
1.  双击该图标。
2.  系统将自动打开三个窗口（后端、客户端、管理端）。
3.  稍等约 15-20 秒，浏览器将自动弹出并打开：
    *   客户端首页: `http://localhost:5173`
    *   管理端后台: `http://localhost:5174/admin/login`

### 方法二：手动脚本启动
在项目根目录下，直接双击运行 **`start_system.bat`** 脚本。

## 3. 开发说明

如果您需要对项目进行开发修改：

*   **修改顾客端页面**：请在 `client/src` 目录下进行。
*   **修改商家端页面**：请在 `admin/src` 目录下进行。
*   **修改后端接口**：请在 `backend/src` 目录下进行。

## 4. 常见问题

*   **端口被占用**：如果启动失败，请检查 `5173`, `5174`, `8080` 端口是否被其他程序占用。
*   **浏览器未自动打开**：请手动访问上述链接。
